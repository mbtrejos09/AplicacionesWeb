Propiedades CSS usadas en To-Do App

Nombre Función Posibles Valores
@import Permite importar archivos CSS externos. Enlace a la hoja de estilos
font-family Define el tipo de fuente del texto. Roboto, Arial, sans-serif, entre otros
Background Establece el color o imagen de fondo. Color en hexadecimal, RGB, imagen
Color Resalta el texto Nombre del color (red), hexadecimal (#ff0000), RGB (rgb(255,0,0))
font-weight Define el grosor del texto. Normal, bold, lighter, bolder, valores numéricos (100 a 900)
border Define un borde alrededor de un elemento. Cborder-width border-style border-color (ej. 1px solid #CECECE)
justify-content Alinea elementos en el eje principal. flex-start, flex-end, center, space-between, space-around
align-items Alinea elementos en el eje secundario (flexbox). flex-start, flex-end, center, stretch
width Define el ancho del elemento. Valores en px, %, auto
padding Define el espacio interno entre el contenido y el borde. Valores en px, em, %, auto
Border Bottom Establece un borde en la parte inferior del elemento. Igual que border, pero solo en la parte inferior
Border Top Establece un borde en la parte superior del elemento. Igual que border, pero solo en la parte superior
Border Radius Redondea las esquinas del borde. Valores en px, %, rem
height Define la altura de un elemento. Valores en px, %, auto
display Determina cómo se muestra un elemento. flex, block, inline, none
max-width Establece el ancho máximo de un elemento. Valores en px, %, none
margin Define el espacio externo alrededor de un elemento. Valores en px, %, none
text-align Alinea el texto dentro de un elemento. left, right, center, justify
